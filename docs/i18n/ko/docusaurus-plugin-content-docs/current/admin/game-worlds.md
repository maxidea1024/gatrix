# 게임 월드

## 개요

게임 서버 인스턴스(월드/채널)를 모니터링하고 관리합니다.

**접근 경로:** 시스템 관리 → 게임 월드

## 기능

- 실시간 서버 상태 확인
- 동시 접속자 수 모니터링
- 월드별 점검 모드 제어
- 서버 수용량 관리

## 월드 상태

| 상태 | 설명 |
|--------|-------------|
| **온라인** | 서버가 정상적으로 실행 중임 |
| **점검 중** | 서버가 점검 모드 상태임 |
| **오프라인** | 서버가 실행 중이 아님 |
| **혼잡** | 서버 수용량이 가득 참 |

## 월드 관리 방법

1. **시스템 관리** > **게임 월드**로 이동합니다.
2. 월드를 클릭하여 상세 정보를 확인합니다.
3. 가능한 동작:
   - 점검 모드 토글
   - 서버 재시작 (플랫폼 연동 시)
   - 접속 중인 플레이어 확인
   - 수용량 조정

## API 연동

SDK를 통해 월드를 등록하고 상태를 업데이트합니다:

```typescript
await gatrix.gameWorlds.register({
  worldId: 'world-1',
  name: '메인 월드 1',
  region: 'KR',
  capacity: 1000,
  currentPlayers: 500,
  status: 'online'
});
```
