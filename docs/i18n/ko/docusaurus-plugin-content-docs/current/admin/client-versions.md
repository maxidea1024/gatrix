# 클라이언트 버전

## 개요

게임 클라이언트 버전 및 업데이트 요구 사항을 관리합니다.

**접근 경로:** 시스템 관리 → 클라이언트 버전

## 기능

- 최소 허용 버전 정의
- 강제 업데이트 알림 설정
- 플랫폼별 버전 관리 (iOS, Android, PC 등)
- 업데이트 URL 구성

## 버전 설정 항목

| 항목 | 설명 |
|-------|-------------|
| **플랫폼** | iOS, Android, Windows, Mac 등 |
| **최소 버전** | 허용되는 가장 오래된 버전 |
| **최신 버전** | 현재 배포된 최신 버전 |
| **강제 업데이트** | 최소 버전 미만일 경우 업데이트 필수 여부 |
| **업데이트 URL** | 스토어 또는 다운로드 링크 |

## 버전 추가 방법

1. **시스템 관리** > **클라이언트 버전**로 이동합니다.
2. **버전 추가** 버튼을 클릭합니다.
3. 다음을 설정합니다:

| 필드 | 유형 | 필수 여부 | 설명 |
|-------|------|----------|-------------|
| 플랫폼 | 선택 | 필수 | 대상 플랫폼 |
| 버전 | 텍스트 | 필수 | 버전 문자열 (예: 1.2.3) |
| 최소 버전 | 텍스트 | 필수 | 최소 요구 버전 |
| 강제 업데이트 | 스위치 | - | 업데이트 안내 강제 여부 |
| 업데이트 URL | 텍스트 | - | 다운로드/스토어 링크 |

4. **저장**을 클릭합니다.

## API 확인

클라이언트는 시작 시 버전을 확인합니다:

```bash
GET /api/v1/client-version?platform=android&version=1.2.0
```

응답 예시:
```json
{
  "needsUpdate": true,
  "forceUpdate": false,
  "latestVersion": "1.3.0",
  "updateUrl": "https://play.google.com/store/apps/..."
}
```
