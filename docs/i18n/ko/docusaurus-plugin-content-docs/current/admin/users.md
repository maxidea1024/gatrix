# 사용자 관리

## 개요

관리자 사용자와 그들의 권한을 관리합니다.

**접근 경로:** 시스템 관리 → 사용자 관리

## 기능

- 관리자 계정 생성
- 역할(Role) 및 권한 할당
- 계정 활성화/비활성화
- 감사 로그 확인

## 사용자 역할(Role)

| 역할            | 설명                         |
| --------------- | ---------------------------- |
| **슈퍼 관리자** | 모든 기능에 대한 전체 권한   |
| **관리자**      | 대부분의 기능에 대한 권한    |
| **운영자**      | 운영 관련 도구에 한정된 권한 |
| **뷰어**        | 읽기 전용 권한               |

## 사용자 생성 방법

1. **시스템 관리** > **사용자 관리**로 이동합니다.
2. **사용자 추가** 버튼을 클릭합니다.
3. 다음을 설정합니다:

| 필드     | 유형     | 필수 여부 | 설명          |
| -------- | -------- | --------- | ------------- |
| 이메일   | 이메일   | 필수      | 로그인 이메일 |
| 이름     | 텍스트   | 필수      | 표시될 이름   |
| 비밀번호 | 비밀번호 | 필수      | 초기 비밀번호 |
| 역할     | 선택     | 필수      | 사용자 역할   |
| 활성화   | 스위치   | -         | 계정 상태     |

4. **생성**을 클릭합니다.

## 비밀번호 정책

- 최소 8자 이상
- 대문자 하나 이상 포함
- 숫자 하나 이상 포함
- 특수 문자 하나 이상 포함

## 2단계 인증 (2FA)

보안 강화를 위해 사용자의 프로필 설정에서 2FA를 활성화할 수 있습니다.
