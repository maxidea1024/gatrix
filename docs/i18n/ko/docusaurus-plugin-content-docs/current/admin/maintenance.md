# 점검 관리

## 개요

서버 점검 시간을 예약하고 관리합니다.

**접근 경로:** 시스템 관리 → 점검 관리

## 기능

- 정기 점검 예약
- 긴급 점검 모드 전환
- 테스터용 화이트리스트 우회 지원
- 사용자 정의 점검 메시지 설정

## 점검 예약 방법

1. **시스템 관리** > **점검 관리**로 이동합니다.
2. **점검 예약** 버튼을 클릭합니다.
3. 다음을 설정합니다:

| 필드 | 유형 | 필수 여부 | 설명 |
|-------|------|----------|-------------|
| 제목 | 텍스트 | 필수 | 점검 제목 |
| 설명 | 텍스트 영역 | - | 상세 설명 |
| 시작 일시 | 날짜/시간 선택 | 필수 | 점검 시작 시각 |
| 종료 일시 | 날짜/시간 선택 | 필수 | 예상 종료 시각 |
| 안내 메시지 | 서식 있는 텍스트 | - | 사용자에게 표시될 메시지 |
| 긴급 여부 | 스위치 | - | 긴급 점검으로 표시 |

4. **생성**을 클릭하여 저장합니다.

## 긴급 점검

긴급한 상황이 발생한 경우, **긴급 점검**을 사용하여 즉시 모든 접근을 차단할 수 있습니다:

1. **긴급 점검** 버튼을 클릭합니다.
2. 동작을 확인합니다.
3. 화이트리스트에 없는 모든 사용자의 접속이 즉시 차단됩니다.

## 화이트리스트 우회

화이트리스트에 등록된 계정이나 IP는 점검 중에도 게임에 접속할 수 있습니다. 자세한 내용은 [화이트리스트](./whitelist)를 참조하세요.

## API 상태 조회

점검 상태를 확인하는 API:

```bash
GET /api/v1/status
```
