# 서비스 공지

## 개요

게임의 서비스 공지사항을 생성하고 관리합니다.

**접근 경로:** 게임 운영 → 서비스 공지

## 기능

- 서비스 공지사항 생성 및 편집
- 게시 시작 및 종료 시간 예약
- 공지 카테고리 분류
- 서식 있는 텍스트(Rich Text) 편집기 지원
- 다국어 지원

## 공지사항 작성 방법

1. **게임 운영** > **서비스 공지**로 이동합니다.
2. **공지 추가** 버튼을 클릭합니다.
3. 다음 양식을 작성합니다:

| 필드 | 유형 | 필수 여부 | 설명 |
|-------|------|----------|-------------|
| 활성화 | 스위치 | - | 공지 표시 여부 토글 |
| 다국어 | 옵션 | - | 다국어 콘텐츠 활성화 |
| 시작 일시 | 날짜/시간 선택기 | - | 공지가 표시되기 시작하는 일시 |
| 종료 일시 | 날짜/시간 선택기 | - | 공지가 만료되는 일시 |
| 카테고리 | 선택 | 필수 | 공지 유형 (공지, 점검, 이벤트 등) |
| 제목 | 텍스트 | 필수 | 공지 제목 |
| 부제목 | 텍스트 | - | 목록 표시에 사용되는 짧은 제목 |
| 내용 | 서식 있는 텍스트 | 필수 | 공지 본문 내용 |
| 설명 | 텍스트 영역 | - | 관리자용 내부 메모 |

4. **생성**을 클릭하여 저장합니다.

## 공지 카테고리

- **공지** - 일반적인 공지사항
- **점검** - 서버 점검 안내
- **이벤트** - 인게임 이벤트
- **업데이트** - 게임 업데이트 및 패치
- **긴급** - 긴급 공지사항

## API 접근

공지사항은 Edge API를 통해 조회할 수 있습니다:

```bash
GET /api/v1/notices
```

자세한 내용은 [클라이언트 API](../api/client-api)를 참조하세요.
