# 배너

## 개요

게임 내에 표시되는 프로모션 배너를 관리합니다.

**접근 경로:** 게임 운영 → 배너

## 기능

- 이미지 배너 생성
- 표시 위치 설정
- 노출 기간 예약
- 클릭 시 동작 구성
- 노출수 및 클릭수 추적

## 배너 생성 방법

1. **게임 운영** > **배너**로 이동합니다.
2. **배너 추가** 버튼을 클릭합니다.
3. 배너를 설정합니다:

| 필드 | 유형 | 필수 여부 | 설명 |
|-------|------|----------|-------------|
| 이름 | 텍스트 | 필수 | 배너 이름 |
| 이미지 | 이미지 업로드 | 필수 | 배너 이미지 |
| 위치 | 선택 | 필수 | 노출 위치 |
| 동작 유형 | 선택 | - | 클릭 시 수행되는 동작 |
| 동작 URL | 텍스트 | - | URL 또는 딥링크 |
| 시작 일시 | 날짜/시간 선택 | - | 노출 시작 일시 |
| 종료 일시 | 날짜/시간 선택 | - | 노출 종료 일시 |
| 우선순위 | 숫자 | - | 표시 순서 |

4. **생성**을 클릭하여 저장합니다.

## 배너 노출 위치

- **메인** - 메인 화면 배너
- **이벤트** - 이벤트 페이지 배너
- **상점** - 상점 페이지 배너
- **로그인** - 로그인 화면 배너

## 동작 유형

- **없음** - 동작 없음
- **URL** - 외부 URL 열기
- **딥링크** - 게임 내 특정 화면 이동
- **공지사항** - 특정 공지 표시
