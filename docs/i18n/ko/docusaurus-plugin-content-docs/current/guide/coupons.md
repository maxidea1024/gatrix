# 쿠폰

## 개요

플레이어를 위한 보상 쿠폰을 생성하고 관리합니다.

**접근 경로:** 게임 운영 → 쿠폰

## 기능

- 고유한 쿠폰 코드 생성
- 사용 제한 설정 (쿠폰당, 사용자당)
- 만료 날짜 구성
- 보상 정의 (아이템, 재화 등)
- 사용 내역 추적

## 쿠폰 생성 방법

1. **게임 운영** > **쿠폰**으로 이동합니다.
2. **쿠폰 추가** 버튼을 클릭합니다.
3. 쿠폰을 설정합니다:

| 필드           | 유형           | 필수 여부 | 설명                    |
| -------------- | -------------- | --------- | ----------------------- |
| 코드           | 텍스트         | 필수      | 고유한 쿠폰 코드        |
| 설명           | 텍스트 영역    | -         | 관리자용 메모           |
| 시작 일시      | 날짜/시간 선택 | -         | 쿠폰이 유효해지는 일시  |
| 종료 일시      | 날짜/시간 선택 | -         | 쿠폰이 만료되는 일시    |
| 최대 사용 횟수 | 숫자           | -         | 전체 사용 제한 횟수     |
| 사용자당 제한  | 숫자           | -         | 사용자당 사용 제한 횟수 |
| 보상           | 보상 설정      | 필수      | 지급할 아이템/재화      |

4. **생성**을 클릭하여 저장합니다.

## 보상 설정

보상에는 다음이 포함될 수 있습니다:

- 인게임 아이템
- 가상 재화
- 프리미엄 시간
- 특별 칭호/뱃지

## 쿠폰 사용

플레이어는 게임 클라이언트를 통해 쿠폰을 사용합니다. API 엔드포인트:

```bash
POST /api/v1/coupons/redeem
{
  "code": "SUMMER2024"
}
```

## 추적

다음을 포함한 사용 내역을 확인합니다:

- 사용 시간
- 사용자 ID
- 지급된 보상
- 상태 (성공/실패)
