# 설문조사

## 개요

사용자 피드백을 수집하기 위해 인게임 설문조사를 생성하고 관리합니다.

**접근 경로:** 게임 운영 → 설문조사

## 기능

- 다양한 질문 유형의 설문 생성
- 특정 사용자 세그먼트 타겟팅
- 설문 수집 기간 설정
- 응답 데이터 분석
- 결과 내보내기

## 설문조사 생성 방법

1. **게임 운영** > **설문조사**로 이동합니다.
2. **설문 추가** 버튼을 클릭합니다.
3. 설문을 설정합니다:

| 필드          | 유형           | 필수 여부 | 설명                 |
| ------------- | -------------- | --------- | -------------------- |
| 제목          | 텍스트         | 필수      | 설문 제목            |
| 설명          | 텍스트 영역    | -         | 설문 설명            |
| 시작 일시     | 날짜/시간 선택 | -         | 설문이 시작되는 일시 |
| 종료 일시     | 날짜/시간 선택 | -         | 설문이 종료되는 일시 |
| 타겟 세그먼트 | 선택           | -         | 타겟 대상            |

4. 질문을 추가합니다.
5. **생성**을 클릭하여 저장합니다.

## 질문 유형

- **객관식 (단일 선택)** - 라디오 버튼, 하나의 답변
- **객관식 (다중 선택)** - 체크박스, 여러 답변
- **주관식 (단답/서술)** - 자유 입력 응답
- **평점** - 1~5점 별점
- **척도** - 숫자 척도 (1~10)

## 응답 분석

다음을 포함한 설문 결과를 확인합니다:

- 총 응답 수
- 응답률
- 답변 분포
- 평균 평점
- CSV로 내보내기
