# 기획 데이터

## 개요

게임 밸런스 및 설정 데이터(기획 데이터)를 관리합니다.

**접근 경로:** 게임 운영 → 기획 데이터

## 기능

- JSON 형식의 설정 데이터 저장
- 데이터 버전 관리
- 환경별 값 설정
- 배포 없이 즉각적인 업데이트

## 주요 활용 사례

- 게임 밸런스 파라미터
- 레벨별 필요 경험치
- 아이템 드랍률
- 이벤트 구성 설정
- 시즌별 밸런스 값

## 기획 데이터 생성 방법

1. **게임 운영** > **기획 데이터**로 이동합니다.
2. **데이터 추가** 버튼을 클릭합니다.
3. 설정을 작성합니다:

| 필드   | 유형        | 필수 여부 | 설명             |
| ------ | ----------- | --------- | ---------------- |
| 키     | 텍스트      | 필수      | 고유 데이터 키   |
| 이름   | 텍스트      | 필수      | 표시될 이름      |
| 설명   | 텍스트 영역 | -         | 관리자용 메모    |
| 데이터 | JSON 에디터 | 필수      | 구성 데이터 내용 |
| 환경   | 선택        | -         | 타겟 환경        |

4. **생성**을 클릭하여 저장합니다.

## 예시

```json
{
  "key": "level_requirements",
  "data": {
    "levels": [
      { "level": 1, "exp": 0 },
      { "level": 2, "exp": 100 },
      { "level": 3, "exp": 300 },
      { "level": 4, "exp": 600 },
      { "level": 5, "exp": 1000 }
    ]
  }
}
```

## API 접근

```bash
GET /api/v1/planning-data/:key
```
