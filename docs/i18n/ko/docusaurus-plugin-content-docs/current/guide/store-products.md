# 상점 상품

## 개요

인앱 구매 상품 및 가상 아이템을 관리합니다.

**접근 경로:** 게임 운영 → 상점 상품

## 기능

- 상품 카탈로그 정의
- 가격 설정 (실제 화폐, 가상 재화)
- 구매 한도 설정
- 가용 기간 예약
- 판매 현황 추적

## 상품 생성 방법

1. **게임 운영** > **상점 상품**으로 이동합니다.
2. **상품 추가** 버튼을 클릭합니다.
3. 상품을 설정합니다:

| 필드 | 유형 | 필수 여부 | 설명 |
|-------|------|----------|-------------|
| 상품 ID | 텍스트 | 필수 | 고유 상품 식별자 |
| 이름 | 텍스트 | 필수 | 표시될 이름 |
| 설명 | 텍스트 영역 | - | 상품 설명 |
| 가격 | 숫자 | 필수 | 가격 (센트/포인트 단위) |
| 화폐 | 선택 | 필수 | USD, KRW, 젬 등 |
| 아이콘 | 이미지 | - | 상품 이미지 |
| 최대 구매 수 | 숫자 | - | 사용자당 구매 제한 |
| 표시 시작 | 날짜/시간 선택 | - | 판매 시작 일시 |
| 표시 종료 | 날짜/시간 선택 | - | 판매 종료 일시 |

4. **생성**을 클릭하여 저장합니다.

## 상품 유형

- **소모품** - 여러 번 구매할 수 있는 아이템
- **비소모품** - 일회성 구매 아이템
- **구독형** - 정기 결제 상품
