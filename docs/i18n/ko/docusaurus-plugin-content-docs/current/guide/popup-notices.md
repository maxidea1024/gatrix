# 팝업 공지

## 개요

플레이어가 로그인할 때나 게임 플레이 중에 나타나는 인게임 팝업 공지를 생성합니다.

**접근 경로:** 게임 운영 → 팝업 공지

## 기능

- 게임 로그인 시 팝업 표시
- 특정 사용자 세그먼트 타겟팅
- 표시 기간 예약
- 이미지 및 서식 있는 콘텐츠 지원
- 조회수 추적

## 팝업 공지 작성 방법

1. **게임 운영** > **팝업 공지**로 이동합니다.
2. **팝업 추가** 버튼을 클릭합니다.
3. 팝업을 설정합니다:

| 필드          | 유형             | 필수 여부 | 설명                           |
| ------------- | ---------------- | --------- | ------------------------------ |
| 활성화        | 스위치           | -         | 팝업 표시 여부 토글            |
| 제목          | 텍스트           | 필수      | 팝업 제목                      |
| 내용          | 서식 있는 텍스트 | 필수      | 팝업 본문 내용                 |
| 이미지        | 이미지 업로드    | -         | 선택적 팝업 이미지             |
| 시작 일시     | 날짜/시간 선택   | -         | 팝업 표시가 시작되는 일시      |
| 종료 일시     | 날짜/시간 선택   | -         | 팝업 표시가 중단되는 일시      |
| 표시 빈도     | 선택             | -         | 한 번만, 매일, 로그인할 때마다 |
| 타겟 세그먼트 | 선택             | -         | 특정 사용자 세그먼트           |

4. **생성**을 클릭하여 저장합니다.

## 표시 빈도 옵션

- **한 번만** - 사용자당 한 번만 표시
- **매일** - 하루에 한 번 표시
- **로그인할 때마다** - 게임 로그인 시마다 표시

## 타겟팅

특정 사용자 세그먼트에게 팝업을 타겟팅할 수 있습니다:

- 신규 사용자
- 복귀 사용자
- VIP 사용자
- 특정 지역/국가
