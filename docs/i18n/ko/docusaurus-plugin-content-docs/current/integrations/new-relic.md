# New Relic 연동

Gatrix의 메트릭과 추적 데이터를 New Relic으로 내보냅니다.

## 설정 방법

### 1. New Relic License Key 확보

1. New Relic에 로그인합니다.
2. **API Keys** 메뉴로 이동합니다.
3. **License Key**를 복사합니다.

### 2. Gatrix에서 구성

1. **설정** > **외부 연동** > **New Relic**으로 이동합니다.
2. 복사한 License Key를 입력합니다.
3. 내보낼 데이터를 선택합니다:
   - 메트릭
   - 사용자 정의 이벤트
   - 피처 플래그 변경 내역
4. **저장**을 클릭합니다.

## 내보낼 데이터

### 메트릭

- API 응답 시간
- 요청 수
- 오류율
- 피처 플래그 평가 횟수

### 사용자 정의 이벤트

- 피처 플래그 변경
- 사용자 활동
- 시스템 이벤트

## 대시보드 활용

연동 후 New Relic 대시보드를 생성하여 다음을 시각화할 수 있습니다:

- 피처 플래그 사용 현황
- 출시 진행률
- 플래그 변경과 시스템 오류 간의 상관관계

## 문제 해결

데이터가 나타나지 않는 경우:

1. License Key가 정확한지 확인하세요.
2. Gatrix 내의 연결 상태를 확인하세요.
3. 초기 데이터가 나타나기까지 약 5분 정도 기다려 보세요.
