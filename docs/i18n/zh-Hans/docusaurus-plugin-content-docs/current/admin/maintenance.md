# 维护管理

## 概述

安排和管理服务器维护时间窗口。

**访问路径：** 系统管理 → 维护管理

## 功能

- 安排周期性维护
- 紧急维护模式切换
- 测试人员白名单绕过
- 自定义维护提示信息

## 安排维护的方法

1. 前往 **系统管理** > **维护管理**。
2. 点击 **安排维护** 按钮。
3. 配置如下：

| 字段     | 类型         | 是否必填 | 说明                   |
| -------- | ------------ | -------- | ---------------------- |
| 标题     | 文本         | 必填     | 维护标题               |
| 说明     | 文本域       | -        | 详细描述               |
| 开始时间 | 日期时间选择 | 必填     | 维护开始时间           |
| 结束时间 | 日期时间选择 | 必填     | 预计结束时间           |
| 提示信息 | 富文本       | -        | 面向玩家显示的公告信息 |
| 紧急维护 | 开关         | -        | 标记为紧急维护         |

4. 点击 **创建** 保存。

## 紧急维护

在紧急情况下，可以使用 **紧急维护** 立即阻止所有访问：

1. 点击 **紧急维护** 按钮。
2. 确认操作。
3. 除白名单外的所有用户将被立即拦截。

## 白名单绕过

白名单中的账号和 IP 可以在维护期间访问游戏。详情请参见 [白名单](./whitelist)。

## API 状态查询

检查维护状态：

```bash
GET /api/v1/status
```
