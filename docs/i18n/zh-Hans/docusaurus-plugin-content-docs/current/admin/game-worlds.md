# 游戏世界

## 概述

监控和管理游戏服务器实例（世界/分线）。

**访问路径：** 系统管理 → 游戏世界

## 功能

- 实时服务器状态
- 在线人数监控
- 单个世界的维护模式切换
- 服务器容量管理

## 世界状态

| 状态 | 说明 |
|--------|-------------|
| **在线** | 服务器运行正常 |
| **维护中** | 服务器处于维护模式 |
| **离线** | 服务器未运行 |
| **爆满** | 服务器人数已达上限 |

## 管理世界的方法

1. 前往 **系统管理** > **游戏世界**。
2. 点击某个世界查看详情。
3. 可用操作：
   - 切换维护模式
   - 重启服务器（如果已集成）
   - 查看当前在线玩家
   - 调整容量上限

## API 集成

通过 SDK 注册及更新世界状态：

```typescript
await gatrix.gameWorlds.register({
  worldId: 'world-1',
  name: 'World 1',
  region: 'KR',
  capacity: 1000,
  currentPlayers: 500,
  status: 'online'
});
```
