# 用户管理

## 概述

管理管理员用户及其权限。

**访问路径：** 系统管理 → 用户管理

## 功能

- 创建管理员账号
- 分配角色和权限
- 启用/禁用账号
- 查看审计日志

## 用户角色

| 角色           | 说明                     |
| -------------- | ------------------------ |
| **超级管理员** | 对所有功能拥有完整权限   |
| **管理员**     | 对大多数功能拥有访问权限 |
| **运营人员**   | 仅限于访问运营相关的工具 |
| **观察者**     | 仅拥有只读权限           |

## 创建用户的方法

1. 前往 **系统管理** > **用户管理**。
2. 点击 **添加用户** 按钮。
3. 配置如下：

| 字段 | 类型     | 是否必填 | 说明         |
| ---- | -------- | -------- | ------------ |
| 邮箱 | 邮箱地址 | 必填     | 登录邮箱     |
| 姓名 | 文本     | 必填     | 显示名称     |
| 密码 | 密码     | 必填     | 初始登录密码 |
| 角色 | 下라选择 | 必填     | 用户角色     |
| 启用 | 开关     | -        | 账号启用状态 |

4. 点击 **创建**。

## 密码策略

- 至少 8 个字符
- 包含至少一个大写字母
- 包含至少一个数字
- 包含至少一个特殊字符

## 双因素认证 (2FA)

用户可以在个人资料设置中启用 2FA 以增强账号安全性。
