# 受众群组

受众群组（Segments）允许您将用户分类，以便针对特定群体展示功能。

## 什么是受众群组？

受众群组是基于特定规则（如用户 ID、国家、等级、VIP 状态等）定义的用户集合。

## 创建受众群组

1. 进入 **Segments** 菜单。
2. 点击 **新建受众群组**。
3. 定义规则：
   - **属性**: `userId`, `country`, `version` 等
   - **操作符**: `equals`, `contains`, `greater than` 等
   - **值**: 匹配的具体值

## 应用场景

- **内部测试**: 将所有内部账号 ID 放入一个群组
- **新用户**: 过去 7 天内注册的用户
- **高价值用户**: 达到一定等级或有过充值的用户
- **特定地区**: 仅针对某个国家或语言的用户
