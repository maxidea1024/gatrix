# 功能开关

功能开关是一种无需重新部署代码即可实时控制功能的工具。

## 概述

功能开关（Feature Flags）允许您在不发布代码的情况下控制功能的启用/禁用，并可以将功能仅暴露给特定的用户群组。

## 主要功能

- **实时切换** - 无需更改代码即可开关功能
- **环境配置** - 针对开发/测试/生产环境分别控制
- **群组定位** - 仅向特定用户群组展示功能
- **A/B 测试支持** - 逐步发布和测试
- **立即回滚** - 发生问题时立即禁用功能

## 创建功能开关

1. 在仪表板中进入 **Feature Flags** 菜单。
2. 点击 **新建功能开关** 按钮。
3. 输入以下信息：
   - **键 (Key)**: 唯一标识符（例如 `new_payment_system`）
   - **名称**: 显示名称
   - **描述**: 功能描述
   - **类型**: 从 Boolean, String, Number, JSON 中选择
