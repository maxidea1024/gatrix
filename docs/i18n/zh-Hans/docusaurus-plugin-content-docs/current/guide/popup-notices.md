# 弹窗公告

## 概述

创建在玩家登录时或游戏过程中显示的弹窗公告。

**访问路径：** 游戏运营 → 弹窗公告

## 功能

- 游戏登录时显示弹窗
- 针对特定用户群组
- 预约显示时间
- 支持图片和富文本内容
- 追踪查看次数

## 创建弹窗公告的方法

1. 前往 **游戏运营** > **弹窗公告**。
2. 点击 **添加弹窗** 按钮。
3. 配置弹窗：

| 字段 | 类型 | 是否必填 | 说明 |
|-------|------|----------|-------------|
| 启用 | 开关 | - | 切换弹窗显示状态 |
| 标题 | 文本 | 必填 | 弹窗标题 |
| 内容 | 富文本 | 必填 | 弹窗正文内容 |
| 图片 | 图片上传 | - | 可选的弹窗图片 |
| 开始时间 | 日期时间选择 | - | 弹窗开始显示的日期和时间 |
| 结束时间 | 日期时间选择 | - | 弹窗结束显示的日期和时间 |
| 显示频率 | 下拉选择 | - | 仅一次、每日、每次登录 |
| 目标群组 | 下拉选择 | - | 特定的用户群组 |

4. 点击 **创建** 保存。

## 显示频率选项

- **仅一次** - 每个用户仅显示一次
- **每日** - 每日显示一次
- **每次登录** - 每次游戏登录时都显示

## 群组定位

您可以针对特定的用户群组投放弹窗：
- 新用户
- 回流用户
- VIP 用户
- 特定地区/国家
