# 商店商品

## 概述

管理应用内购买商品和虚拟道具。

**访问路径：** 游戏运营 → 商店商品

## 功能

- 定义商品目录
- 设置定价（真实货币、虚拟货币）
- 配置购买限制
- 预约上架和下架时间
- 追踪销售情况

## 创建商品的方法

1. 前往 **游戏运营** > **商店商品**。
2. 点击 **添加商品** 按钮。
3. 配置商品：

| 字段       | 类型         | 是否必填 | 说明                  |
| ---------- | ------------ | -------- | --------------------- |
| 商品 ID    | 文本         | 必填     | 唯一的商品标识符      |
| 名称       | 文本         | 必填     | 显示名称              |
| 说明       | 文本域       | -        | 商品描述              |
| 价格       | 数字         | 必填     | 价格（以分/点为单位） |
| 货币       | 下拉选择     | 必填     | USD, KRW, Gems 等     |
| 图标       | 图片         | -        | 商品图片              |
| 最大购买数 | 数字         | -        | 每用户购买限制        |
| 开始时间   | 日期时间选择 | -        | 开始销售时间          |
| 结束时间   | 日期时间选择 | -        | 停止销售时间          |

4. 点击 **创建** 保存。

## 商品类型

- **消耗品** - 可以多次购买的道具
- **非消耗品** - 一次性购买道具
- **订阅** - 周期性订阅商品
