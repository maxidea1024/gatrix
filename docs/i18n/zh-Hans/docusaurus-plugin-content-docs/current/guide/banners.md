# 广告横幅

## 概述

管理游戏内显示的推广横幅。

**访问路径：** 游戏运营 → 广告横幅

## 功能

- 创建图片横幅
- 设置显示位置
- 预约投放周期
- 配置点击动作
- 追踪曝光和点击

## 创建横幅的方法

1. 前往 **游戏运营** > **广告横幅**。
2. 点击 **添加横幅** 按钮。
3. 配置横幅：

| 字段 | 类型 | 是否必填 | 说明 |
|-------|------|----------|-------------|
| 名称 | 文本 | 必填 | 横幅名称 |
| 图片 | 图片上传 | 必填 | 横幅图片 |
| 位置 | 下拉选择 | 必填 | 显示位置 |
| 动作类型 | 下拉选择 | - | 点击时的动作 |
| 动作 URL | 文本 | - | URL 或深层链接 |
| 开始时间 | 日期时间选择 | - | 开始投放时间 |
| 结束时间 | 日期时间选择 | - | 停止投放时间 |
| 优先级 | 数字 | - | 显示顺序 |

4. 点击 **创建** 保存。

## 横幅位置

- **主页** - 主屏幕横幅
- **活动** - 活动页面横幅
- **商店** - 商店页面横幅
- **登录** - 登录屏幕横幅

## 动作类型

- **无** - 无动作
- **URL** - 打开外部网页
- **深层链接** - 游戏内导航
- **公告** - 显示特定公告
