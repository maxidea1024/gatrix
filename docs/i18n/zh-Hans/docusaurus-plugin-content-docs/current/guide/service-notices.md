# 服务公告

## 概述

创建和管理游戏的系统公告。

**访问路径：** 游戏运营 → 服务公告

## 功能

- 创建和编辑服务公告
- 预约发布和过期时间
- 公告分类管理
- 支持富文本编辑器
- 多语言支持

## 创建公告的方法

1. 前往 **游戏运营** > **服务公告**。
2. 点击 **添加公告** 按钮。
3. 填写以下表单：

| 字段 | 类型 | 是否必填 | 说明 |
|-------|------|----------|-------------|
| 启用 | 开关 | - | 切换公告显示状态 |
| 多语言 | 选项 | - | 启用多语言内容 |
| 开始时间 | 日期时间选择器 | - | 公告开始显示的日期和时间 |
| 结束时间 | 日期时间选择器 | - | 公告过期的日期和时间 |
| 分类 | 下拉选择 | 必填 | 公告类型（通知、维护、活动等） |
| 标题 | 文本 | 必填 | 公告标题 |
| 副标题 | 文本 | - | 用于列表显示的简短标题 |
| 内容 | 富文本 | 必填 | 公告的正文内容 |
| 说明 | 文本域 | - | 管理员内部备注 |

4. 点击 **创建** 保存。

## 公告分类

- **通知** - 一般性公告
- **维护** - 服务器维护通知
- **活动** - 游戏内活动
- **更新** - 游戏更新和补丁
- **紧急** - 紧急通知

## API 访问

可以通过 Edge API 获取公告：

```bash
GET /api/v1/notices
```

详情请参见 [客户端 API](../api/client-api)。
