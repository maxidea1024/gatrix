# 优惠券

## 概述

为玩家创建和管理奖品优惠券。

**访问路径：** 游戏运营 → 优惠券

## 功能

- 生成唯一的优惠券代码
- 设置使用限制（每张优惠券、每个用户）
- 配置过期日期
- 定义奖励（道具、货币等）
- 追踪兑换历史

## 创建优惠券的方法

1. 前往 **游戏运营** > **优惠券**。
2. 点击 **添加优惠券** 按钮。
3. 配置优惠券：

| 字段 | 类型 | 是否必填 | 说明 |
|-------|------|----------|-------------|
| 代码 | 文本 | 必填 | 唯一的优惠券代码 |
| 说明 | 文本域 | - | 管理员备注 |
| 开始时间 | 日期时间选择 | - | 优惠券开始生效的日期和时间 |
| 结束时间 | 日ี期时间选择 | - | 优惠券过期的日期和时间 |
| 最大使用次数 | 数字 | - | 总兑换限制次数 |
| 每用户限制 | 数字 | - | 每个用户兑换限制次数 |
| 奖励 | 奖励配置 | 必填 | 发放的道具/货币 |

4. 点击 **创建** 保存。

## 奖励配置

奖励可以包括：
- 游戏内道具
- 虚拟货币
- 高级会员时长
- 特殊称号/勋章

## 优惠券兑换

玩家通过游戏客户端兑换优惠券。API 接口：

```bash
POST /api/v1/coupons/redeem
{
  "code": "SUMMER2024"
}
```

## 追踪

查看兑换历史，包括：
- 兑换时间
- 用户 ID
- 发放的奖励
- 状态（成功/失败）
