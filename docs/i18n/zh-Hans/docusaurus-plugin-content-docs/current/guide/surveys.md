# 问卷调查

## 概述

创建和管理游戏内问卷，以收集玩家反馈。

**访问路径：** 游戏运营 → 问卷调查

## 功能

- 创建包含多种题型的问卷
- 针对特定用户群组
- 设置调查周期
- 分析调研数据
- 导出结果

## 创建问卷的方法

1. 前往 **游戏运营** > **问卷调查**。
2. 点击 **添加问卷** 按钮。
3. 配置问卷：

| 字段 | 类型 | 是否必填 | 说明 |
|-------|------|----------|-------------|
| 标题 | 文本 | 必填 | 问卷标题 |
| 说明 | 文本域 | - | 问卷描述 |
| 开始时间 | 日期时间选择 | - | 问卷开始时间 |
| 结束时间 | 日期时间选择 | - | 问卷关闭时间 |
| 目标群组 | 下拉选择 | - | 目标群组 |

4. 添加问题。
5. 点击 **创建** 保存。

## 题型

- **单选题** - 单选按钮，一个答案
- **多选题** - 复选框，多个答案
- **文本题** - 开放式文本回答
- **评分题** - 1-5 星级评分
- **量表题** - 数字量表 (1-10)

## 结果分析

查看问卷结果，包括：
- 总回复数
- 回复率
- 答案分布
- 平均评分
- 导出为 CSV
