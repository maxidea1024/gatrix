# 策划数据

## 概述

管理游戏平衡和配置数据（游戏策划数据）。

**访问路径：** 游戏运营 → 策划数据

## 功能

- 存储 JSON 格式的配置数据
- 配置文件版本控制
- 针对不同环境的数值设置
- 无需发布即可即时更新

## 使用场景

- 游戏数值平衡参数
- 等级经验需求
- 道具掉落率
- 活动配置
- 季节性设置

## 创建策划数据的方法

1. 前往 **游戏运营** > **策划数据**。
2. 点击 **添加数据** 按钮。
3. 配置如下：

| 字段     | 类型        | 是否必填 | 说明           |
| -------- | ----------- | -------- | -------------- |
| 键 (Key) | 文本        | 必填     | 唯一的数据键名 |
| 名称     | 文本        | 必填     | 显示名称       |
| 说明     | 文本域      | -        | 管理员备注     |
| 数据     | JSON 编辑器 | 必填     | 配置数据内容   |
| 环境     | 下拉选择    | -        | 目标环境       |

4. 点击 **创建** 保存。

## 示例

```json
{
  "key": "level_requirements",
  "data": {
    "levels": [
      { "level": 1, "exp": 0 },
      { "level": 2, "exp": 100 },
      { "level": 3, "exp": 300 },
      { "level": 4, "exp": 600 },
      { "level": 5, "exp": 1000 }
    ]
  }
}
```

## API 访问

```bash
GET /api/v1/planning-data/:key
```
