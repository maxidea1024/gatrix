# Game Engine SDKs

Gatrix provides SDKs specifically optimized for popular game engines to enable seamless feature flagging within your game projects.

## Overview

Game Engine SDKs are tailored to handle the unique requirements of game development:
- Support for engine-specific languages (C++, C#, GDScript, Luau, GML).
- Efficient flag evaluation optimized for game loops (60+ FPS).
- Cross-platform support for consoles, PC, and mobile.

## Key Features

- **Local Evaluation**: Zero network latency during game execution.
- **Resource Management**: Minimal memory footprint and no allocations during evaluation.
- **Real-time Updates**: Reflect flag changes instantly without restarting the game.
