# Gatrix 시작하기

## 기능 설명

Gatrix는 온라인 게임 플랫폼을 위한 관리 시스템입니다.

### 제공 기능

**게임 운영:**
- 공지사항 (Service Notices)
- 인게임 팝업 (Popup Notices)
- 쿠폰 (Coupons)
- 배너 (Banners)
- 설문조사 (Surveys)
- 상점 상품 (Store Products)
- 기획 데이터 (Planning Data)

**시스템 관리:**
- 클라이언트 버전 (Client Versions)
- 게임 월드 (Game Worlds)
- 점검 관리 (Maintenance)
- 사용자 관리 (Users)

**API:**
- Client API
- Server SDK API
