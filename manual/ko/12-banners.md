# 제 13장: 배너 관리 (Banner Management)

게임 로비, 상점 등에 노출되는 배너 그룹(롤링 배너)을 관리합니다.

**경로:** 게임 (Game) → 배너 (Banners)  
**필요 권한:** `banners_manage`

---

## 1. 배너 등록 및 시퀀스

하나의 배너 항목 안에 여러 개의 이미지를 **시퀀스(Sequences)** 형태로 등록할 수 있습니다. 이는 애니메이션 효과나 롤링 배너를 구현할 때 사용됩니다.

1. **[+ 배너 생성]** 클릭
2. **기본 정보:** 이름, 설명, 크기(Width/Height) 입력
3. **시퀀스 추가:** 실제로 노출될 이미지들을 순서대로 등록합니다.

---

## 2. 상태 관리 및 배포 (Publish)

배너는 세 가지 상태를 가집니다.

- **Draft (작성 중):** 아직 배포되지 않은 상태
- **Published (배포됨):** 실제 게임에 적용되어 유저에게 보이는 상태
- **Archived (보관됨):** 사용이 종료되어 보관된 상태

### 배포하기
작성이 완료된 배너는 **Action Menu (⋮)** → **[Publish]**를 눌러야 유저에게 노출됩니다.

### 보관하기
더 이상 사용하지 않는 배너는 **Action Menu (⋮)** → **[Archive]**를 눌러 보관 처리합니다.

---

## 3. 복제 (Duplicate)

비슷한 배너를 새로 만들 때 유용합니다.
**Action Menu (⋮)** → **[Duplicate]**를 선택하면 이름 뒤에 `(copy)`가 붙은 똑같은 배너가 생성됩니다.
